# UNICIVE - POLO SETE LAGOAS

Bem-vindo ao repositório do site da Unicive Polo Sete Lagoas!

## 🌐 Páginas Disponíveis

- **[🏠 Site Principal](./index.html)** - Site oficial com cursos e informações
- **[📋 To-Do List](./todo-list.html)** - Aplicação de gerenciamento de tarefas
- **[🌤️ Weather Dashboard](./weather-dashboard.html)** - Previsão do tempo em tempo real

## 📁 Estrutura

```
├── index.html              # Site principal
├── todo-list.html          # Aplicação To-Do List
├── weather-dashboard.html  # Dashboard de Clima
└── README.md               # Este arquivo
```

## ✨ Recursos

### Site Principal
- Apresentação dos cursos (Graduação, Pós-Graduação, Tecnólogo, Técnico)
- Seção sobre o Polo
- Diferenciais
- Formulário de contato
- Menu responsivo

### To-Do List
- Adicionar, editar e deletar tarefas
- Filtros (Todas, Ativas, Concluídas)
- Prioridades (Alta, Média, Baixa)
- Datas opcionais
- Local Storage - dados persistem
- Estatísticas em tempo real

### Weather Dashboard
- Busca por cidade
- Clima atual com detalhes
- Previsão 5 dias
- API OpenWeatherMap
- Design responsivo

## 🎨 Design

- Cores coordenadas: Verde primário (#02634C) e Laranja secundário (#FF8C00)
- Totalmente responsivo
- Animações suaves
- Interface moderna e intuitiva

## 🚀 Como Usar

Todos os arquivos são HTML puros, sem necessidade de compilação.

1. Clone o repositório
2. Abra qualquer arquivo `.html` no navegador
3. Ou acesse online via GitHub Pages

## 📝 Tecnologias

- HTML5
- CSS3
- JavaScript Vanilla
- OpenWeatherMap API
